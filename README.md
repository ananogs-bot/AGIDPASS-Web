# AGID PASS - Landing Page

Landing page oficial do projeto **AGID PASS**, desenvolvida para apresentar a plataforma, seus objetivos, funcionalidades, tecnologias utilizadas e direcionar usuários para a experiência mobile.

## 📖 Sobre o Projeto

O AGID PASS é uma plataforma voltada para o segmento de beleza e bem-estar, conectando clientes e profissionais em um ambiente digital moderno e intuitivo.

A solução foi desenvolvida com o objetivo de facilitar o acesso a serviços de autocuidado, permitindo a divulgação de profissionais, consulta de serviços e gerenciamento de agendamentos.

Esta landing page funciona como portal institucional do projeto, apresentando suas funcionalidades, tecnologias, equipe de desenvolvimento e links para os demais artefatos produzidos.

---

## 🚀 Funcionalidades da Landing Page

- Apresentação institucional do projeto
- Explicação do funcionamento da plataforma
- Demonstração dos serviços disponíveis
- Apresentação das tecnologias utilizadas
- Integração com o AgidBot (Botpress)
- Área de cadastro  integrada ao Firebase
- Divulgação dos links oficiais do projeto
- Design responsivo para desktop e dispositivos móveis

---

## 📱 Aplicativo Mobile

A experiência principal do AGID PASS foi desenvolvida para dispositivos móveis utilizando Android Studio.

Pelo aplicativo é possível:

- Realizar cadastro de clientes e profissionais
- Gerenciar perfil
- Consultar catálogo de profissionais
- Visualizar detalhes dos serviços
- Realizar agendamentos
- Utilizar o chatbot AgidBot
- Acompanhar informações da conta

## 🤖 AgidBot

O AgidBot é o assistente virtual integrado ao projeto, desenvolvido utilizando Botpress.

Seu objetivo é auxiliar usuários com:

- Dúvidas sobre a plataforma
- Navegação no sistema
- Busca por serviços
- Informações sobre funcionalidades
- Suporte básico ao usuário
